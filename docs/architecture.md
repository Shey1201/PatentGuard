## 三、核心模块设计

### 3.1 知识库模块 (Knowledge Base)

**职责**: 管理文档的导入、存储、向量化

**核心流程**:
```
用户上传文档 → 文件存储 (MinIO) → 文本提取 (parser) 
→ 分块 (chunking) → 向量化 (embedding) → 存入向量库
```

**关键服务**:
- `parser.py`: 支持 PDF、DOCX、TXT 解析
- `embedding.py`: 调用 Embedding API 生成向量
- `storage.py`: MinIO 文件管理

### 3.2 检索模块 (Retriever)

**职责**: 从知识库中检索相关内容

**核心流程**:
```
用户文档 → 生成 Query Embedding → 向量相似度检索 (pgvector)
→ 关键词检索 (BM25) → 融合排序 → 返回 Top-K 结果
```

**优化策略**:
- 混合检索：向量 + 关键词
- 重排序：使用交叉编码器优化结果

### 3.3 LLM 分析模块 (Analysis Engine)

**职责**: 基于检索结果进行合规审查

**Prompt 设计**:
```
你是专业的专利/文档合规审查专家。

请基于以下法规和知识库内容，对用户文档进行分析：

【知识库内容】
{retrieved_context}

【用户文档】
{user_document}

请输出 JSON 格式结果：
{
  "compliance": true/false,
  "risk_level": "low/medium/high",
  "summary": "总体评价",
  "findings": [
    {
      "type": "risk/suggestion",
      "severity": "low/medium/high",
      "description": "问题描述",
      "reference": "法规依据",
      "suggestion": "修改建议"
    }
  ]
}
```

### 3.4 系统配置模块

**职责**: 允许用户配置 LLM API

**支持 providers**:
- OpenAI (GPT-4, GPT-3.5)
- 阿里云通义千问
- Anthropic Claude
- 智谱清言
- 其他 OpenAI 兼容接口

### 4.2 异步处理设计

对于大文档审查，采用异步处理：

```
POST /api/v1/analysis/review
  ↓
创建任务 → 返回 task_id
  ↓
轮询: GET /api/v1/analysis/tasks/{task_id}
  ↓
任务完成 → 获取结果
```

## 四、安全设计

### 4.1 认证

- JWT Token 认证
- Token 有效期：24 小时
- 刷新机制

### 4.2 权限

| 操作 | 管理员 | 普通用户 |
|------|--------|----------|
| 管理知识库 | ✅ | ❌ |
| 上传审查文档 | ✅ | ✅ |
| 查看结果 | ✅ | ✅ |
| 配置 LLM | ✅ | ❌ |
| 删除文档 | ✅ | ❌ |

### 4.3 API 安全

- 请求频率限制 (Rate Limiting)
- 输入验证 (Pydantic)
- SQL 注入防护 (SQLAlchemy ORM)
- XSS 防护

## 五、可扩展性设计

### 5.1 模块化架构

- 核心逻辑与服务解耦
- 易于替换 Embedding/LLM Provider
- 插件化支持新的文档类型

### 5.2 水平扩展

- Docker Compose 支持多实例
- 可接入 Kubernetes
- 数据库读写分离 (未来)

### 5.3 缓存策略

- LLM 响应缓存 (相同问题)
- Embedding 缓存 (相同文本)
- Redis 缓存层

## 六、技术选型总结

| 组件 | 选型 | 理由 |
|------|------|------|
| 后端框架 | FastAPI | 高性能、易用、异步支持 |
| 数据库 | PostgreSQL + pgvector | 免费、易部署、支持向量索引 |
| 文件存储 | MinIO | S3 兼容、本地部署 |
| 前端 | React + TypeScript | 主流、现代、类型安全 |
| 缓存 | Redis | 高性能、丰富功能 |
| 向量索引 | pgvector (HNSW) | 集成在 Postgres 中、效率高 |
